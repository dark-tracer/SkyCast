# Weather Dashboard

A web application that displays real-time weather updates for a user's location using a public weather API.

## Features

* Current weather conditions
* 7-day forecast
* Responsive charts
* Search by city

## Tech Stack

* HTML
* CSS
* JavaScript (React)
* Fetch API
* OpenWeatherMap API
